# ankush1529.github.io

Personal website of **Ankush Kr. Mandal** — Software Engineer at Microsoft.

Live at **https://ankush1529.github.io**.

Built with [Jekyll](https://jekyllrb.com/) on the
[Pudhina](https://github.com/Knhash/Pudhina) theme.

## Run locally

```bash
bundle install      # first time only
jekyll serve        # then open http://127.0.0.1:4000
```

## Structure

- `index.md` — main page (about, education, experience, publications).
- `_config.yml` — site settings (title, URL, social links).
- `_layouts/`, `_includes/` — Jekyll templates.
- `assets/` — CSS, images, fonts, and the downloadable résumé.
